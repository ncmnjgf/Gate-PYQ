================================
TROUBLESHOOTING
================================

If you still see errors:

ERROR: "Client ID not found"
SOLUTION:
- Double-check .env file name (must be ".env" not ".env.txt")
- Make sure no spaces around = sign
- Restart the development server

ERROR: "Gemini API key not found"
SOLUTION:
- Verify REACT_APP_GEMINI_API_KEY is in .env file
- No quotes around the API key
- Restart the development server

ERROR: "Failed to load resource: net::ERR_NAME_NOT_RESOLVED"
SOLUTION:
- Check your internet connection
- Try different network if using VPN/proxy
- Disable antivirus temporarily

ERROR: "403 Forbidden" on Google Sign-In
SOLUTION:
- The Google Client ID might need to be updated
- Go to: https://console.cloud.google.com/apis/credentials
- Add http://localhost:3000 to authorized origins
- Add http://localhost:3000 to authorized redirect URIs
